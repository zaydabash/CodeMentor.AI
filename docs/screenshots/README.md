# Screenshots

This directory contains screenshots of the CodeMentor.AI interface.

## Required Screenshots

Please add the following screenshots to this directory:

1. **landing-page.png** - Screenshot of the main landing page with hero section and feature cards
2. **upload-interface.png** - Screenshot of the upload page showing ZIP/Git URL options
3. **analysis-dashboard.png** - Screenshot of the job analysis page with issues list and filters
4. **pr-viewer.png** - Screenshot of the PR draft viewer with diffs and export options

## Taking Screenshots

1. Start the development servers: `make dev`
2. Navigate through the application
3. Take screenshots at 1920x1080 or similar resolution
4. Save them with the names listed above in this directory
5. The README.md will automatically display them
